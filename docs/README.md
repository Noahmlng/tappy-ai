# Docs 分类索引

1. `design/`：产品与系统设计规范（定义是什么、为什么、边界是什么）。
2. `execution-plans/`：阶段计划与落地执行文档（定义先后顺序、验收标准、推进路径）。
3. `other/`：接入说明、运行手册、项目结构等参考资料（服务于使用与排障）。
