### 3.8 Module F: Event & Attribution Processor

#### 3.8.1 事件合同（当前最小集）

事件最小集冻结：
1. `impression`
2. `click`
3. `failure`

必填语义：
1. `responseReference`
2. 事件类型
3. 事件时间
4. 状态与原因码（适用时）
5. 渲染事件到 M1 的映射规则见 `3.7.24`（必须保持一致）。

#### 3.8.2 处理规则

1. 事件必须先归一再归因。
2. 无 `responseReference` 事件进入隔离轨道，不进标准口径。
3. 事件窗口超时时系统补写 `failure` 终态，保证闭环可完成。

#### 3.8.3 输出合同

1. 事件归一记录与关联结果。
2. 闭环终态更新信号。
